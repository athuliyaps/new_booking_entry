# New Booking Entry - Logistics ERP

A responsive New Booking Entry screen for a Logistics ERP system built as a machine task using React and TypeScript.

This screen allows office administrators to quickly enter sender, receiver, and package details with validations and automatic cost calculation.

---

##  Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Hot Toast

---
